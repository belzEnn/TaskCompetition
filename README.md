# TaskCompetition

TaskCompetition — це інтерактивна платформа, яка перетворює нудні хатні справи на змагання. Користувачі об'єднуються у віртуальні кімнати, де кожен отримує власне випадкове завдання. 

Ваша задача накормити свиню швидше ніж ваш соперник. Ваша свиня не такі як всі свині, вона любить чистоту і порядок.

[python 3.14.3](https://www.python.org/downloads/release/python-3143/),
requirements:
```
pip install -r requirements
```
